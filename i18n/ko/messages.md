# Learn Quest 한국어 메시지 템플릿

이 파일은 `language: "ko"` 설정 시 사용되는 한국어 메시지 템플릿입니다.

## 설정 메뉴 (Config)

```
🎮 LEARN QUEST 설정

현재 설정:
• 레벨: [현재 레벨]
• 자동 설명: [ON/OFF]
• 패시브 모드: [ON/OFF] (빈도: [자동/높음/보통/낮음])
• 정보 제공: [ON/OFF]
• 방향성 제시: [ON/OFF]
• CS 지식: [ON/OFF]
• 퀴즈: [ON/OFF]
• 언어: [en/ko]
• 저장 위치: [local/project]

무엇을 변경할까요?
1) 레벨 변경 (Bronze/Silver/Gold/Platinum/Diamond)
2) 자동 설명 ON/OFF
3) 패시브 모드 설정
4) 기능별 ON/OFF 설정
5) 언어 변경
6) 저장 위치 변경
7) 설정 초기화

> 번호를 입력하세요
```

## 설명 모드 선택 (Explain)

```
🎮 LEARN QUEST

무엇에 대해 알고 싶으세요?

1) 방금 작업한 코드 설명
2) 직접 입력할게요

> 번호를 선택하세요
```

```

## 패시브 모드 출력

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎮 LEARN QUEST (패시브)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔄 더 생각해볼 점
- [핵심 인사이트]

💡 관련 개념
- [CS 개념 간략 설명]

📝 퀵 체크: [간단한 질문]
```

## 패시브 모드 설정

```
🎯 패시브 학습 모드를 활성화할까요?

패시브 모드는 질문할 때 자동으로 학습 요소를 추가합니다.
단순히 답변만 받는 게 아니라:
- 핵심 인사이트와 고려사항
- 관련 CS 개념
- 간단한 퀴즈 (활성화된 경우)

추가 노력 없이 코딩하면서 자연스럽게 배울 수 있어요.

1) 네, 패시브 모드 활성화 (권장)
2) 아니요, /learn-quest:explain으로 직접 학습할게요
```

## 빈도 선택

```
📊 학습 요소가 얼마나 자주 나타날까요?

1) 자동   - 레벨과 학습량에 따라 조절 (권장)
2) 높음   - 거의 모든 학습 기회에
3) 보통   - 적당한 빈도로
4) 낮음   - 드물게, 중요한 개념만
```
